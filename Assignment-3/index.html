JavaScript String Questions

Q1. Count Characters

Given a string S, return an array B where:
B[0] = count of 'A', B[1] = count of 'D'.
Note: Complete countCharacters().

Q2. Count the Heads

Count characters that occur more than once in string S. Return in alphabetical order with their count.
Note: Complete Count_Head().

Q3. Count the Vowels

Find the number of vowels in string S.
Note: Complete Count_Vowel().

Q4. Concatenate the Strings

Concatenate two strings S1 and S2.
Note: Complete Concatenate_Strings().

Q5. Find Length

Find length of string S without using .length.
Note: Complete findLength().

Q6. Find the Winner

String S contains 'A' and 'D'. Each represents a win. Return winner or Draw.
Note: Complete Game_Winner().

Q7. Join Strings

Concatenate two space-separated strings.
Note: Complete joinStrings().

Q8. Palindrome Check

Check if string S is a palindrome.
Note: Complete Plain_Check().

Q9. Reverse the String

Reverse the given string S.
Note: Complete Reverse_String().

Q10. Match the Strings

Print YES if both strings are the same, else NO.
Note: Complete String_Match().

Q11. String Replace

Replace a given pattern in string S with text.
Note: Complete Replace().

Q12. Split the String

Split string S by spaces and return array.
Note: Complete Split_the_String().

Q13. Count Vowels and Consonants

Return number of vowels and consonants in string S.
Note: Complete Count_Vowels() & Count_Consonants().
