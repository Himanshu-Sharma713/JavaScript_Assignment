JavaScript Loop Assignment

Q1. Print Pattern

Write a JavaScript program to construct the following pattern using a nested for loop:
*
* *
* * *
* * * *
* * * * *

Q2. FizzBuzz

Iterate through numbers from 1 to 45. Print:
- "Fizz" for multiples of 3
- "Buzz" for multiples of 5
- "FizzBuzz" for multiples of both

Q3. Sum of Array

Calculate the sum of all numbers in the array:
[1, 13, 22, 123, 49]

Q4. Sum of Two Arrays

Given arrays:
[3, 5, 22, 5, 7, 2, 45, 75, 89, 21, 2]
[9, 2, 42, 55, 71, 22, 4, 5, 90, 25, 26]
Output the total sum of their elements.
Example: 276 + 351 = 627

Q5. Reverse Array

Print the elements of the array in reverse order:
[43, "what", 9, true, "cannot", false, "be", 3, true]
